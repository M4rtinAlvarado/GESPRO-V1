## 1. Requisitos Funcionales (RF)
| ID | Nombre | Descripción |
| :--- | :--- | :--- |
| **RF-01** | **Importación de Carta Gantt** | El sistema debe permitir la creación de un proyecto mediante la carga de un archivo Excel, validando que la estructura coincida con la plantilla oficial. |
| **RF-02** | **Gestión de Alertas Automáticas** | El sistema debe enviar correos electrónicos automáticos a los encargados cuando se aproxime la fecha límite de una tarea o cambie su estado a "En progreso". |
---

## 2. Requisitos No Funcionales (RNF)
Estos definen **cómo** debe ser el sistema (propiedades, restricciones o cualidades).

| ID | Nombre | Descripción |
| :--- | :--- | :--- |
| **RNF-01** | **Usabilidad (Interfaz)** | La interfaz debe ser intuitiva, permitiendo que el usuario acceda a un proyecto desde el menú principal en no más de dos clics. |
| **RNF-02** | **Compatibilidad de Archivos** | El módulo de exportación debe generar archivos en formato `.xlsx` compatibles con versiones modernas de Microsoft Excel, manteniendo el formato original. |
